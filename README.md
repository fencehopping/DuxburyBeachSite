# Duxbury Beach Site

Simple static marketing site for the Duxbury Beach app.

## Pages

- `index.html` - app marketing homepage
- `privacy.html` - privacy policy
- `terms.html` - terms of service

## Local Preview

```sh
python3 -m http.server 4174
```

Then open `http://127.0.0.1:4174/index.html`.

## Deployment

This site has no build step. Deploy the repository root to any static host, including GitHub Pages, Netlify, Vercel, or Cloudflare Pages.
